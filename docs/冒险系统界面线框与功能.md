# 冒险系统界面线框与功能

## 1. 界面概述
冒险系统是游戏的核心主玩法模块。玩家在此推进主线关卡、触发战斗、获取放置挂机收益。

## 2. 界面线框布局
```text
+-------------------------------------------------+
| 玩家头像 Lv.45   [冒险] 主线 12-4    [战力: 12w] |
+-------------------------------------------------+
|                                                 |
|               (动态战斗演出区域)                    |
|        [敌方单位]     VS     [我方女仆队伍]        |
|                                                 |
+-------------------------------------------------+
|   挂机收益: 💰 1234  💎 5  📦 12                |
|                  [ 领取收益 ]                    |
+-------------------------------------------------+
| [快速战斗]         [首领挑战]           [阵容配置] |
+-------------------------------------------------+
| [商店]     [升级]     [冒险]     [装备]     [女仆] | <- 全局底部导航(选中高亮)
+-------------------------------------------------+
```

## 3. 核心功能区域
1. **沙盘/战斗表现层**：上半屏为持续进行的挂机战斗表现（或3D跑图），实时展现角色释放技能清怪的动画。
2. **放置收益区**：随时间推移积累奖励，点击【领取收益】将奖励收入囊中（宝箱最多累计24小时）。
3. **核心操作栏**：
   - **首领挑战**：挂机进度充满后，点击挑战当前关卡BOSS。胜利后进入下一关。
   - **快速战斗**：消耗资源直接获取120分钟的挂机收益。
   - **阵容配置**：调整上场的女仆站位和出战策略。

## 4. 交互流程
1. 在底部导航点击【冒险】切回主舞台。
2. 玩家观察挂机产生的大量金币、装备提示。
3. 点击【领取收益】，弹出全屏奖励结算弹窗。
4. 点击【首领挑战】，屏幕切换至Boss战斗场景，若胜利则关卡进度推进至"12-5"，若失败则弹出提升战力的指引（引导前往升级或装备界面）。
