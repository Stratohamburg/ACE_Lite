# 商店系统界面线框与功能

## 1. 界面概述
商店系统除了常规的资源和道具售卖，还包含**抽卡（招募）系统**。为了增加趣味性，核心的抽卡系统表现为**“盲盒贩卖机”**的形式。玩家在这里获取新的女仆和碎片。

## 2. 界面线框布局

### 2.1 综合商店布局
```text
+-------------------------------------------------+
| [返回]               商店               [金币][钻石]   |
+-------------------------------------------------+
| [女仆贩卖机] [每日精选] [道具商城] [礼包中心] [兑换所] | <- 二级页签
+-------------------------------------------------+
| ... (常规商品列表网格) ...                         |
+-------------------------------------------------+
```

### 2.2 贩卖机（抽卡）布局
当选中【女仆贩卖机】页签时，界面中央切出贩卖机的互动区域：

|                                                 |
|    +=======================================+    |

|    |   [ ? ] [ ? ] [ ? ] [ ? ] [ ? ]       |    |

## 3. 核心功能区域
1. **顶部资源栏**：显示玩家当前的钻石、金币以及专用的【贩卖机代币】。
2. **二级子商店标签**：其中【女仆贩卖机】页签具有特殊的视觉权重（可带常驻光效或Live2D特效）。
3. **贩卖机交互区**：
   - 包含玻璃橱窗表现，里面陈列着盲盒/胶囊/饮料罐作为抽卡包装。
   - 显示下方的出口和当前的保底计数器（如“X抽后必得SSR”）。
   - 提供单抽（投币1次）和十连抽（投币10次）按钮。
4. **常规商品展区**：其他页签依然为常规的卡片式商品陈列。

## 4. 交互流程（贩卖机抽卡流程）
1. 玩家点击【女仆贩卖机】页签，屏幕中央展示一个精美的3D/2D动态贩卖机。
2. 玩家点击【投币 1次】或【投币 10次】。
3. **演出阶段**：播放金币掉入投币口的音效，随后机器内的商品（盲盒/胶囊）掉落至出货口，并冒出不同颜色的光芒（蓝光=R，紫光=SR，金光=SSR）。
4. **结算阶段**：弹出抽取结果面板，展示抽到的女仆立绘及评级。如果抽到已有的女仆，自动转化为角色碎片。
5. 点击【确认识别】，返回贩卖机界面。
