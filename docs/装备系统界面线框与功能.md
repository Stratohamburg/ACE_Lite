# 装备系统界面线框与功能

## 1. 界面概述
装备系统用于管理玩家（或女仆队伍）获取的各种战斗装备，包括装备的穿戴、强化、精炼与分解回收。

## 2. 界面线框布局
```text
+-------------------------------------------------+
| [返回]               装备               [筛选/排序]|
+-------------------------------------------------+
|                                                 |
|  [武器]  [头盔]  [衣服]  [鞋子]  [饰品]  [神石]    |
|                                                 |
|  当前选中: 【破败之刃】 Lv.20                     |
|  稀有度: 史诗 (紫色)                              |
|  属性: 攻击力 +1200, 暴击率 +5%                   |
+-------------------------------------------------+
|                    |  装备背包                  |
|  [ 强化 ]          |  [衣] [武] [武] [头]       |
|  消耗: 8000金币    |  [鞋] [饰] [饰] [衣]       |
|  [ 一键分解 ]      |  [ ]  [ ]  [ ]  [ ]       |
|                    |                           |
+-------------------------------------------------+
| [商店]     [升级]     [冒险]     [装备]     [女仆] | <- 全局底部导航
+-------------------------------------------------+
```

## 3. 核心功能区域
1. **装备槽位预览**：上方展示当前已穿戴的各个部位装备图标，可点击查看详细对比。
2. **装备属性详情与操作**：中部显示当前选中装备的数值情况。提供【强化】、【精炼】等养成按钮。
3. **装备背包网格**：右侧或下方提供滑动的背包列表，展示当前持有的未穿戴装备。提供筛选（按部位、按品质）功能。
4. **分解系统**：对于低级无用的装备，可通过一键分解转化为强化材料。

## 4. 交互流程
1. 玩家点击底部导航【装备】进入。
2. 在背包中选中一件新获取的强力装备，界面左侧出现与当前穿戴装备的属性对比（绿色上升箭头表示提升）。
3. 点击【一键穿戴】或【替换】，装备更换。
4. 点击【强化】，显示动画并扣除金币，属性数值跳跃上升。若强化满级则提示需突破/进阶。
