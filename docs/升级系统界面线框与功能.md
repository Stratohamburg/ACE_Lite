# 升级系统界面线框与功能

## 1. 界面概述
升级系统主要负责玩家主角（或团队整体）的基础属性、技能树提升以及全局增益效果的激活。

## 2. 界面线框布局
```text
+-------------------------------------------------+
| [返回]               升级               [金币][钻石] |
+-------------------------------------------------+
|    主角等级: Lv 45           战力: 125,000       |
|  [||||||||||||||||||---] (经验条进度)           |
+-------------------------------------------------+
|                   [ 主角立绘/模型 ]                 |
|                                                 |
+-------------------------------------------------+
| [属性强化]    | 攻击力  + 1500 (下一级: 1650)      |
| [技能树]      | 生命值  + 8000 (下一级: 8800)      |
| [全局天赋]    | 防御力  + 400  (下一级: 450)       |
|              |                           [升级] |
+-------------------------------------------------+
| [商店]     [升级]     [冒险]     [装备]     [女仆] | <- 全局底部导航
+-------------------------------------------------+
```

## 3. 核心功能区域
1. **角色属性概览**：展示当前等级、战力、经验值以及模型。
2. **三级页签切换**：
   - **属性强化**：使用基础资源（金币/经验）直接提升基础三维属性。
   - **技能树**：消耗特定天赋点解锁和升级主动/被动技能。
   - **全局天赋**：提供对于产出、掉落等系统层面的加成收益。
3. **升级操作区**：预览升级效果与消耗资源，提供“升级”或“一键升级”按钮。

## 4. 交互流程
1. 点击底部【升级】进入。
2. 右下角显示当前拥有物资与“升级”所需物资。若资源足够，按钮高亮。
3. 点击【升级】按钮，扣减资源，进度条上涨或等级提升，播放提升特效并跳字展示属性增加。
4. 等级提升可能解锁新的女仆栏位或装备槽位，触发相应的新手引导或高亮提示。
